About The Cohort

Cohort 3 is focused on HTML, CSS, JavaScript fundamentals, debugging, and team collaboration using Git and GitHub. We will also cover practical frontend topics like React, TypeScript basics, UI libraries, API integration, performance optimization, testing, and deployment.

Monthly Schedule

  • Month 1: Semantic HTML + accessibility basics
  • Month 2: CSS layouts and responsive design
  • Month 3: JavaScript DOM manipulation
  • Month 4: APIs and async JavaScript
  • Months 5-12: React, TypeScript basics, state management, testing, deployment, and project practice

FAQ

How do I submit assignments?

Push your code to GitHub and share the repo link before the deadline.

What if npm install fails?

Delete node_modules and lock file, then run install again and share the error log if it persists.

Can I miss a live class?

Yes, but watch the recording and complete the catch-up task before next class.
